Solución problemas claves ssh con Git

Esto me ha funcionado en un Mac OS X versión 10.6.8. Iba embalado con el tutorial de Git, que fácil, cuando me encontré un problema con las claves ssh. En esta página explico como lo resolví. Parte 2: Error Publickey ssh

Hastá ahí todo bien pero al llegar aquí:

git push -u origin master me daba un error con las claves ssh En esta página encontré la solución y funcionó! ¡ya reconoce las claves!

Parte 3: Tuve que crear repositorio en github

funcionó pero al repetir el comando git push -u origin master me daba un error: ERROR: Repository not found.

tuve que crear el nombre del repositorio desde GitHub, voler a ejecutar git push -u origin master y entonces funcionó bien.
